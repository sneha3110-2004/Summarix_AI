# Σ SUMMaRIX AI

### Neural Research Intelligence for Engineers

## Summary

SUMMaRIX AI is a full-stack AI-powered research paper summarizer designed to help students and engineers quickly understand complex academic PDFs.
The system uses transformer-based NLP models to generate summaries, explanations, and a chat-based interface for document queries, all running locally without paid API access.
It features a modern glassmorphism dashboard with secure history management and responsive UI.

---

## 🚀 Key Features

* AI-powered research paper summarization (T5 model)
* Chat with uploaded PDF
* Simple language explanation generator
* Key points & notes generator
* Pre-analysis file preview
* Local history storage with download option
* Modern Glassmorphism UI with Dark / Light mode
* No paid API required (fully local NLP)

---
